PetHero

Proyecto final de carrera

PetHero es un proyecto colaborativo (junto con Catalina Liste y Lautaro Manzo) desarrollado con PHP y MySQL para crear una conexión entre dueños de mascotas y guardianes, dónde los dueños pueden poseer una lista de mascotas y crear reservas en un rango de fechas y horario, contando además con un sistema de mensajería entre los mismos.

Mercado Totola

Proyecto cuatrimestral

Este trabajo final de cuatrimestre desarrollado con Java fue presentado en conjunto con Lautaro Manzo y Tobias Valentini. El mismo es una simulación de un supermercado online en el que el usuario posee la capacidad de agregar productos a un "carrito" virtual y proceder a finalizar una compra. También pueden acceder al programa usuarios como gerentes o empleados, para autogestionar sus datos.

Proyecto cuatrimestral
